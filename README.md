# 📅 College Timetable Generator

A simple web-based mini project that allows students to generate a conflict-free timetable by selecting faculty for different subjects and time slots.

---

## 🚀 Features

* 🔐 User Registration & Login (LocalStorage based)
* 📊 Generate Timetable for Sets (A, B, C)
* 👨‍🏫 Faculty Selection (No duplicate allowed)
* ⚡ Real-time validation (Already selected faculty disabled)
* 🖨️ Clean Printable Timetable
* 🎨 Simple and user-friendly UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* LocalStorage (for authentication)
